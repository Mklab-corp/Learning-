💡 Linux Beginner Quick Tips

1
Press TAB for autocomplete
Type cd /ho and press Tab → it completes to /home/
2
Use man command for help
Example: man rsync shows everything about the rsync command
3
Pipe commands with |
tail -f log.txt | grep error → live error monitoring
4
Ctrl + C stops any running command
Super useful when something is taking too long
5
Use history to see past commands
Type !123 to rerun command number 123
6
Always use -h or --help
Most commands show usage when you add these flags
7
Be extremely careful with rm -rf
It permanently deletes everything without asking
8
sudo = superuser do
Use it only when you really need admin rights
